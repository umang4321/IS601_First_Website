Assignment 2: Git Command Line and First Website

Purpose

  • The purpose of the page is to learn basic Linux commands, Vi Text Editor, and Git from the command Line.


    • 1. Linux Commands

  • pwd - Show current directory
  • ls - Directory listing
  • cd - Change directory
  • cd dir - Change directory to dir
  • mkdir dir - Create a directory dir
  • clear - Clears the current window
  • history - Lists the complete history of commands used
  • rm filename - Removes the selected file
  • rm -rf foldername - Removes the selected folder


    • 2. VI Editor Commands

  • i - Insert mode
  • Escape - Get out of insert mode
  • :w - Command to write
  • :q - Command to quit
  • :wq - Write and quit


    • 3. Git Commands

  • git init - Create the repository
  • git add filename - Add the file(s) to git in preparation for a commit
  • git commit -m "message"- Do the commit to save changes
  • git checkout -b branch name - Make a new branch and checkout
  • git checkout branch name - Checkout branch
  • git merge branchname - Merge the branch
